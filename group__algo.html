Execute a functor over each tuple element.

Parameters
[in]tupinput tuple
[in]funuser provided functor

Invoke a functor passing a reference to each tuple element. Functor should accept a generic parameter type and/or handle each type contained in the tuple.

auto tup = make_tuple(1, "foo");
algo::for_each(
tup,
[](auto const& e) {
cout << e << ", ";
}
);

Change type and/or value of each tuple element.

Parameters
[in]tupinput tuple
[in]funmutating function

Translates every tuple element according to provided functor in such a way that new tuple element have type of decltype(fun(tuple_element)) and a value of fun(tuple_element). A good example is to transform a tuple of integers into tuple of references to the original tuple elements.

auto t = std::make_tuple(1, 2.3);
auto t2 = algo::mutate(t, make_reference);
static_assert(is_same_v<decltype(t2), tuple<int &, double &>>);
